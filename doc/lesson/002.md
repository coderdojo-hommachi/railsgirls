Heroku
======

## Get Heroku

[quickstart guide](https://devcenter.heroku.com/start)

### Toolbelt

```
$ heroku login
```

## Production

### Gem

`Gemfile`:

```
gem 'sqlite3', group: :development

group :production do
  gem 'pg'
  gem 'rails_12factor'
end
```

### Bundle

```
$ bundle install --without production
```

----

## Deploy

```
$ heroku create
$ git push heroku master
$ heroku run rake db:migrate
$ heroku open
```
